Version with current nav.

Version with tabbed navigation. This streamlines the top of the page and makes it easier to completely immerse users in maps data.
Version 3 - Geoshapes
Support for geoshapes. Current implementation has geoshape visibility coded by location type (country, state, city, neighborhood, etc.) for zoom level.)
Version 4 - Big20 Extractions
Support for extractions. Does not yet allow for editing points, or for moving points between layers.
